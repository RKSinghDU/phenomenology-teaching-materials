# Activity 07 · Lifeworld Mapping

**Format:** Individual &nbsp;|&nbsp; **Topic:** Method &nbsp;|&nbsp; **Time:** 25 minutes

---

## Description

Students select a significant but non-traumatic personal transition (moving city, starting university, losing a friendship) and map how each of the six lifeworld dimensions was affected — before, during, and after.

---

## Step-by-Step

1. Choose a significant personal transition that is *not* traumatic.
2. Complete the lifeworld mapping grid below.
3. Review: which dimensions were most profoundly disrupted? Which recovered most quickly?
4. Write one sentence: what was the *central* disruption of your lifeworld in this transition?
5. Optional: share one dimension with a partner.

---

## Lifeworld Mapping Grid

| Dimension | Before | During | After |
|-----------|--------|--------|-------|
| **Spatiality** (sense of home, safety, familiarity) | | | |
| **Temporality** (sense of continuity, future, narrative) | | | |
| **Embodiment** (bodily comfort, ease, self-consciousness) | | | |
| **Intersubjectivity** (trust, belonging, being understood) | | | |
| **Mood / Attunement** (background emotional orientation) | | | |
| **Narrative Identity** (sense of who you are, life story coherence) | | | |

---

## Materials

The grid above (can be printed or drawn by hand).

---

## Debrief Guidance

Key points to draw out:

- **Interconnection of dimensions:** Embodiment shifts affect mood; disrupted intersubjectivity affects narrative identity. The dimensions are aspects of a unified lifeworld structure, not separate modules.
- **Comparison to trauma:** In traumatic disruption, the damage is more total, more violent, and the 'after' does not restore the 'before.'
- **Clinical application:** Could you redesign a clinical assessment tool around these six dimensions? What would it capture that standard symptom measures (PHQ-9, PCL-5) miss?

This activity also demonstrates the *phenomenological productivity* of systematic description: simply filling in the grid often produces insights students did not have before they started writing.
