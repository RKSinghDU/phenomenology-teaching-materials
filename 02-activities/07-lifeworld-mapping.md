# Activity 07 · Lifeworld Mapping

**Format:** Individual | **Topic:** Method | **Time:** 25 minutes

---

## Description

Students select a significant but non-traumatic personal transition (moving city, starting university, losing a friendship, changing career) and map how each of the six lifeworld dimensions was affected — before, during, and after the transition.

---

## Step-by-Step Instructions

1. Choose a significant personal transition that is *not* traumatic — something that genuinely disrupted your ordinary lifeworld but that you can reflect on with some distance.
2. Complete the lifeworld mapping grid below for your transition.
3. Review your grid: which dimensions were most profoundly disrupted? Which recovered most quickly? Were any dimensions unchanged?
4. Write one sentence: what was the *central* disruption of your lifeworld in this transition?
5. Optional: share one dimension (not necessarily the most personal) with a partner.

---

## Lifeworld Mapping Grid

| Dimension | Before the transition | During the transition | After the transition |
|-----------|----------------------|-----------------------|----------------------|
| **Spatiality** (Lived space — sense of home, safety, familiarity) | | | |
| **Temporality** (Lived time — sense of continuity, future, narrative) | | | |
| **Embodiment** (Lived body — bodily comfort, ease, self-consciousness) | | | |
| **Intersubjectivity** (Relationships — trust, belonging, being understood) | | | |
| **Mood / Attunement** (Background emotional orientation to the world) | | | |
| **Narrative Identity** (Sense of who you are, coherence of your life story) | | | |

## Materials

Lifeworld mapping grid (above — can be printed or drawn by hand).

## Debrief Guidance

This activity builds the analytical vocabulary before applying it to clinical phenomena. Discussing non-traumatic disruptions first allows students to understand the dimensions with less emotional charge — before encountering them in the context of severe trauma.

Key points to draw out in discussion:

- **Interconnection of dimensions:** Embodiment shifts affect mood; disrupted intersubjectivity affects narrative identity; foreclosed temporality affects spatial experience. The dimensions are not separate — they are aspects of a unified lifeworld structure.
- **Comparison to trauma:** In traumatic disruption, the damage is more total, more violent, and the 'after' does not restore the 'before.' The disruption is not a temporary dislocation but a permanent reorganisation of the lifeworld.
- **Clinical application:** Ask — could you redesign a clinical assessment tool around these six dimensions? What would it look like? What would it capture that standard symptom measures (PHQ-9, PCL-5) miss?

This activity also demonstrates to students the *phenomenological productivity* of systematic description: simply filling in the grid often produces insights students did not have before they started writing.
